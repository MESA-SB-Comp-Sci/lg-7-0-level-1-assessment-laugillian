## Question 1 — Describe

What is the difference between a static unit like `px` and a responsive unit like `%` or `vh`?

Describe what makes a unit responsive and why that matters when building a website.

The difference between a static unit like px and a responsive unit like % or vh is that static doesnt change the size and responsive does change based on screen size. What makes a unit responsive is that it can change on the screen and that matters because if there isnt a responsive unit, the layout on the website can break on different devices and will look weird.

## Question 2 — Explain

Look at these two CSS rules:

```css
.image {
  width: 400px;
}

.image {
  width: 50%;
}
```

Explain what happens to the image on a small screen with each rule. Why does one behave better than the other?

For the first one, the image on a small screen will break and will look odd since its a static unit so it cant change its size. The 2nd one behaves better because its using a responsive unit aka % which means that it will be able to change its size depending on the screen size.
